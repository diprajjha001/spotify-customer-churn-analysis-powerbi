# 🎧 Music Streaming Customer Churn Analysis Dashboard (Power BI)

An interactive **Power BI dashboard** that analyzes music streaming user behavior, engagement patterns, and customer churn to uncover actionable insights for improving user retention and platform performance.

---

## 📊 Project Overview

This project focuses on understanding **why users churn and how they interact with a music streaming platform**.  
Using Power BI, the dashboard visualizes user activity, demographic trends, device usage, network types, and subscription plans to help businesses identify high-risk user segments and optimize retention strategies.

---

## 🎯 Project Objective

The goal of this project is to build an **interactive analytics dashboard** that helps businesses:

- Monitor **customer churn and retention**
- Analyze **user engagement patterns**
- Identify **high-risk segments likely to churn**
- Support **data-driven decision making**

---

## 📌 Key Metrics

- **Total Users:** 589  
- **Active Users:** 411  
- **Churned Users:** 178  
- **Churn Rate:** 30.22%

---

## 📈 Dashboard Insights

### 1️⃣ Customer Churn Overview
Tracks overall platform health using key metrics such as:
- Total Users
- Active Users
- Churned Users
- Churn Rate

---

### 2️⃣ User Demographics Analysis
Analyzes churn and engagement based on:
- Gender
- Age Groups

Helps identify **which demographic segments are most likely to churn**.

---

### 3️⃣ Device & Network Insights
Visualizes user activity across:

**Devices**
- Android
- iOS
- Web

**Network Types**
- WiFi
- 4G
- 5G

This helps identify potential **technical or accessibility factors affecting churn**.

---

### 4️⃣ Music Genre Engagement
Analyzes listening behavior across different genres to understand:
- Which genres drive the **highest engagement**
- Which user groups prefer **specific content types**

---

### 5️⃣ Subscription Plan Analysis
Compares churn trends across:

- Premium Plan
- Family Plan
- Free Plan

This helps determine **which pricing models retain users more effectively**.
